# Student Academic Risk Management System

## Project Overview

The Student Academic Risk Management System is a web-based application developed using PHP and MySQL.The Purpose of this system is to manage student academic information and identify students who may be at academic risk based on their performance data.

This system allows administrators to store and manage studnt records including attendance,internal marks,semester marks,and CGPA .Based on these details,the system calculates the academic risk level of students and classifies them into Low Risk,Medium Risk,High Risk categories.

## Features

- User Login Authentication
- Dashboard with quick access options
- Add Student Records
- View Student Details
- Edit Student Information
- Delete Student Record
- Search Student Data
- Filter Student Records
- Academic Risk Calculation
- Total Students Count Display

## Technologies Used

- PHP
- MySQL
- HTML
- CSS
- XAMPP Servar

## How the System Works

1.User logs in to the system.
2.Dashboard opens with options like Add Student, View Student and Logout.
3.Admin can add student details including:
 - Name
 - Register Number
 - Department
 - Year
 - Attendance
 - Internal Marks
 - Semester Marks
 - CGPA
4.Based on the entered data, the system calculates the student's academic risk level.

## Risk Classification Example

1.High Risk - Low attendance and Low marks
2.Medium Risk - Average performance
3.Low Risk - Good academic performance

## Purpose of the Project

This project was developed to demonstrate basic student data management and academic performance analysis. It helps identify students who may need academic support at an early stage.
